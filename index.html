Current Work

Our research combines the disciplines of cyber-security, machine learning, and cryptography. Working with a graduate student, I am responsible for encrypting plaintext datasets for our deep neural network to be implemented over using a homomorphic encryption library, HElib. Our goal is to see the networks' ability to be implemented over training datasets that it doesn't have direct access to, and to compare the networks' effectiveness to a network trained with data it has direct access to. We believe that if a network can be trained using encrypted data, we can improve the level of security and privacy users would have over their own data from machine learning algorithms, such as Google's search engine.
